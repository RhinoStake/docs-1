# 2. Configure your project

Your entire project is configured with a single configuration file, `config.json`.

A sample `config.json`:

```
{
  "baseTokenURI": "",
  "numTokens": 0,
  "contractCodeId": 0,
  "unitPrice": 1.0,
  "name": "Collection Name",
  "symbol": "SYM",
  "minter": "stars1...",
  "contractURI": "stars1...",
  "creator": "stars1...",
  "royaltyAddress": "stars1...",
  "royaltyShare": 0.05
}
```
